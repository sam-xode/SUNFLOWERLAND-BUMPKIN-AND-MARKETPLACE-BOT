## üîç Command Details

### `/check_bumpkin [farm_id]`

Check detailed information about any Sunflower Land farm by providing its Farm ID.

**Usage:** `/check_bumpkin farm_id:873735225663547`

After using this command, you'll receive an interactive response with buttons to navigate through different sections:

#### Available Sections:
- **Bumpkin Data**: View Bumpkin level, skills, and attributes
- **Resources Data**: Check all resources and their quantities
- **Inventory**: Browse through all items in the farm's inventory
- **Collection**: See collectibles, rare items, and trophies

Each section provides detailed information with proper formatting and categorization:

### `/sfl_calculator [item] [amount]`

Calculate the current SFL value of your in-game items and resources.

**Usage:** `/sfl_calculator item:carrot amount:125`
